Что не нужно маркировать: полный список исключений

Какие товары освобождены от обязательной маркировки. Исключения по категориям, условия освобождения, документальное подтверждение.

8 мин чтения17 декабря 2025 г.

Общие исключения для всех категорий

Следующие товары не подлежат маркировке независимо от категории:

1. Образцы и пробы для испытаний

Товары, предназначенные для проведения испытаний, исследований и сертификации.

Условия:

  • Должны быть документы, подтверждающие цель использования
  • Не подлежат продаже конечным потребителям

    • 2. Экспонаты выставок и ярмарок

    Товары для демонстрации на международных выставках и ярмарках.

    Условия:

  • Наличие документов об участии в мероприятии
  • После окончания выставки товар должен быть вывезен или промаркирован

    • 3. Возвращённые товары

    Товары, возвращённые покупателями продавцу.

    Важно: При повторной продаже требуется перемаркировка.

    4. Товары в магазинах беспошлинной торговли (Duty Free)

    Товары, реализуемые в зонах беспошлинной торговли.

    5. Товары под таможенным контролем

    Товары в режиме таможенного транзита, хранения на таможенном складе.

    Условие: До выпуска в свободное обращение.

    6. Товары для собственных нужд

    Товары, используемые производителем для собственных производственных нужд.

    Исключения по категориям

    Молочная продукция

  • Молочная продукция весом нетто до 20 г (с 01.12.2023)
  • Детское питание для детей до 3 лет (отдельные сроки маркировки)
  • Молочная продукция в общепите (готовые блюда)

    • Табачная продукция

  • Сигары и сигариллы ручной работы
  • Табак для кальяна в развес (без потребительской упаковки)

    • Лекарственные препараты

  • Препараты для клинических исследований
  • Радиофармацевтические препараты
  • Лекарства, изготовленные аптеками по рецепту

    • Обувь и одежда

  • Изделия ручной работы при соблюдении условий:

    • - Единичное производство

    - Отсутствие серийного выпуска

    - Продажа без посредников

  • Бывшая в употреблении одежда/обувь (секонд-хенд)

    • Парфюмерия

  • Пробники и тестеры (не для продажи)
  • Парфюмерия в упаковке менее 3 мл

    • Шины

  • Восстановленные шины (при соблюдении условий)
  • Шины для спецтехники (некоторые коды ТН ВЭД)

    • Велосипеды

  • Детские велосипеды, классифицируемые как игрушки

    • БАДы

  • Участники в отдалённых местностях имеют увеличенный срок регистрации (30 дней)

    • Как подтвердить право на исключение

    Необходимые документы:

  • 1Договоры — на поставку образцов, участие в выставке
  • 2Акты — о возврате товара, о списании
  • 3Сертификаты — подтверждение ручной работы
  • 4Таможенные документы — для транзитных товаров

    • Где хранить:

  • В бухгалтерии организации
  • Срок хранения — не менее 3 лет
  • Предоставлять по требованию контролирующих органов

    • Частые ошибки

    Ошибка 1: «Мелкие партии не маркируются»

    Неверно. Объём партии не влияет на обязанность маркировки. Даже 1 единица товара должна быть промаркирована.

    Ошибка 2: «Продажа через интернет — исключение»

    Неверно. Онлайн-продажи подлежат маркировке наравне с офлайн.

    Ошибка 3: «Товары для личного использования»

    Частично верно. Только если товар не продаётся и не передаётся третьим лицам.
